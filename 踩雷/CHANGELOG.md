# 更新日志 — 踩雷

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)

---

## [1.2.0] — 2026-03-29

### 新增
- 响应式布局：标题、按钮、信息栏等在小屏自动缩放
- 格子尺寸根据屏幕宽度动态计算（困难模式30列也能放下）
- 触屏插旗模式按钮（替代右键，触屏设备自动显示）
- 窗口 resize 时自动重新计算格子尺寸

---

## [1.1.0] — 2026-03-25

### 新增
- 游戏大厅返回链接（左上角固定显示）
- Web Audio API 音效系统：插旗/取旗音效、翻格音效、游戏结束音效、通关音效、破纪录上升音调
- 全局静音按钮，M 键快捷切换
- 三档难度各自的最快时间 localStorage 存储（`小游戏_踩雷_最快时间_简单/中等/困难`）
- 最快时间面板，三档难度并排显示
- 总局数统计（`小游戏_踩雷_总局数`），显示在信息栏
- 破纪录横幅动画（仅当前难度创纪录时触发）
- Enter / R 键快捷重开游戏
- 帮助说明折叠块

### 改动
- localStorage 新增时间记录，原版无任何 localStorage
- 字体补充 `Segoe UI` + `Microsoft YaHei` 优先级
- 网页标题改为「踩雷 — 小游戏合集」
- 主题色从红色（`#e94560`）改为警戒黄（`#ffd54f`），更符合扫雷游戏气质
- 标题颜色跟随主题色发光效果

## [1.0.0] — 2026-03-20

### 新增
- 游戏初始版本（源文件 `踩雷游戏.html`）
- DOM 网格扫雷游戏
- 三档难度：简单 9×9/10 雷、中等 16×16/40 雷、困难 16×30/99 雷
- 第一格安全保证（地雷不落在首击格及其 8 邻格）
- 洪水填充自动展开（零格递归）
- 右键插旗 / 取旗
- 双击已翻数字快速展开（弦操作）
- 计时器（首击后开始）
- 1-8 数字色彩编码
- 踩雷震动动画
