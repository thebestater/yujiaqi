## [1.1.0] — 2026-03-25

### 新增
- 游戏大厅返回链接（左上角固定显示）
- Web Audio API 音效系统：吃食物音效、游戏结束音效、破纪录上升音调
- 全局静音按钮，状态跨游戏共享
- 总局数统计（`小游戏_贪吃蛇_总局数`），显示在信息栏
- 破纪录横幅动画
- P 键暂停（与空格键等效）
- 触屏滑动手势支持
- 帮助说明折叠块

### 改动
- localStorage key 从 `snakeHighScore` 统一为 `小游戏_贪吃蛇_最高分`
- 字体补充 `Segoe UI` 优先级
- 网页标题改为「贪吃蛇 — 小游戏合集」
- 控制栏新增暂停按钮，状态与快捷键同步

### 修复
- 暂停状态下画布不再重复绘制，减少 CPU 占用

## [1.0.0] — 2026-03-19

- 游戏初始版本（源文件 `snake.html`）

- 空格键暂停
- 速度随长度加快（最快 50ms/步）
- 触屏方向按钮（窄屏自动显示）
