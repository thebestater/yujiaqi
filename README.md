# 佳琪的小游戏合集

> 迷你游戏集合， 打开游戏大厅即可开始游戏。

## 快速开始

1. 双击 `index.html` — 打开游戏大厅
2. 点击任意游戏卡片 — 进入游戏
3. 游戏内左上角点击「← 游戏大厅」返回

## 开发新游戏

1. 复制 `_template/` 文件夹，重命名为游戏中文名（如 `五子棋/`）
2. 参考模板内注释填写游戏逻辑
3. 在 `index.html` 的 `GAMES` 数组中追加一个配置对象
4. 更新本 README 的游戏列表
5. 补充更新日志

**开发前置清单**：每款新游戏开始前，先与用户确认 A–F 六组信息（见 `_template/index.html` 顶部注释）。

## 通用按键

| 按键 | 功能 |
|---|---|
| 空格 / P | 暂停 / 继续 |
| Enter | 开始 / 重开 |
| R | 重新开始 |
| M | 静音切换 |
| Escape | 暂停或返回 |

## 技术规范

- **格式**：单文件 HTML + 内嵌 CSS/JS，无外部依赖，完全离线可用
- **音效**：Web Audio API 代码合成，无需音频文件
- **存储**：localStorage，key 格式为 `小游戏_[游戏名]_[字段]`
- **兼容**：Edge / Chrome / Firefox（Windows 桌面）
