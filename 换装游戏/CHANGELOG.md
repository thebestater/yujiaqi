# 更新日志 — 换装小屋

所有重要改动都记录于此文件。

---

## [1.1.0] — 2026-03-29

### 新增
- 手机端（≤480px）响应式布局：标题缩放、角色卡片紧凑化、服装网格3列适配
- 换装室/场景选择在小屏自动调整间距和尺寸
- 允许页面垂直滚动（修复 overflow:hidden 导致小屏内容截断）

---

## [1.0.1] — 2026-03-26

### 修复
- 修复游戏完全无法启动的严重 Bug：内部辅助函数 `top()` 与浏览器全局属性 `window.top` 同名，导致脚本抛出 `Identifier 'top' has already been declared` 错误，所有游戏逻辑函数失效；已将该函数重命名为 `makeTop()` 并同步更新所有调用处

---

## [1.0.0] — 2026-03-25

### 新增
- 初始版本发布
- 3 个可选角色：小米（黑发）、艾拉（金发）、小星（紫发魔法精灵）
- 30 件服装：7 连衣裙 + 5 上装 + 5 下装 + 4 帽饰 + 3 发饰 + 3 包包 + 3 鞋子
- 8 个场景：公主城堡、魔法森林、海边沙滩、春日公园、生日派对、咖啡小屋、太空星球、水下世界
- 每个场景含 1-2 个可点击装饰物（互动动画）
- 角色可执行挥手 / 跳跃 / 转圈 三种动作
- Web Audio API 音效系统（7 种音效）
- localStorage 记录最近造型与场景，下次打开自动恢复
- 共享全局静音状态 `小游戏_全局_静音`
- 全 SVG 矢量绘制，无外部资源依赖
