## [1.2.0] — 2026-03-29

### 新增
- 响应式布局：手机端三栏自动堆叠为纵向排列
- 触屏虚拟按键组（⬅ 🔄 ➡ ⬇ ⏬），触屏设备自动显示
- 小屏适配：面板紧凑化，按键说明在手机端隐藏
- 极小屏（≤360px）画布自动缩放

## [1.1.0] — 2026-03-25

### 新增
- 游戏大厅返回链接（左上角固定显示）
- Web Audio API 音效系统：落块音效、消行音效（消 4 行更响亮）、游戏结束音效、破纪录上升音调
- 音效开关按钮（右面板），M 键快捷切换，全局静音状态共享
- 总局数统计（`小游戏_俄罗斯方块_总局数`），显示在左面板
- 破纪录横幅动画
- 暂停时画布渲染暂停遮罩文字

### 改动
- localStorage key 从 `tetris_best` 统一为 `小游戏_俄罗斯方块_最高分`
- 背景修复为三段渐变（`#1a1a2e → #16213e → #0f3460`），与其他游戏一致
- 字体补充 `Segoe UI` + `Microsoft YaHei` 优先级
- 网页标题改为「俄罗斯方块 — 小游戏合集」
- 面板背景改用统一 `rgba(255,255,255,0.05)` 令牌

### 修复
- 暂停期间 `requestAnimationFrame` 仍运行（不影响逻辑，已修复渲染重叠）

## [1.0.0] — 2026-03-22

### 新增
- 游戏初始版本（源文件 `tetris.html`）
- Canvas 俄罗斯方块，10×20 标准格
- 7 种标准 Tetrimino（I/O/T/L/J/S/Z），对应原版颜色
- 幽灵块（落点预览）
- 墙踢旋转系统（偏移 0/1/-1/2/-2）
- 消行计分：1-100 / 2-300 / 3-500 / 4-800，乘以等级
- 软降 +1 / 硬降 +2 加成
- 等级自动提升（每 10 行），下落速度加快
- 最高分 localStorage 存储
- 下一块预览面板
- P 键暂停
